What is it?

In the simplest of terms, virtualization allows for the simulation of a computer within a computer. Virtualization allows for multiple virtual machines (VMs) to be ran on a single server. Since physical servers aren't always using 100% of their given resources, efficiency and other economic factors are not being optimized for. For example, instead of having 3 physical servers each using 30% of their resources, we can virtualize and create 3 virtual machines on a single server, allowing for 90% utilization of resources.
