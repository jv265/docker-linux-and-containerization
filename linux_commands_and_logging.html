Basic Linux Commands

  • pwd - prints the current directory you are in
  • ls - lists all the files in your current directory
  • cd - change to another directory
  • mkdir - create a directory
  • rmdir - remove an empty directory
  • rm - remove a file, or remove a directory with files (rm -r)
  • touch - create an empty file
  • man - display a manual page of a command
  • cp - copy a file
  • echo - display the commands argument back into the terminal, can also be redirected into a file
  • cat - display the contents of a file

Linux Docker Commands

  • docker --version - display docker version
  • docker search nginx - search for docker images (e.x. nginx) from a registry (default to Docker Hub)
  • docker pull nginx - download docker image with latest tag (default) from registry
    • docker pull nginx:1.19.3 - download a specific version (tag) of an image
  • docker push myrepo/myimage:tagname - push changes of your image to a repository
  • docker images - list all downloaded docker images
    • docker image rm nginx - delete an image from the local image store
  • docker run --name nginx-container -it nginx - run a docker container from the nginx image. -it will allow for an interactive shell and allocate a tty. From the user perspective, this is running a shell inside the container. The --name flag will assign the container with the supplied name, otherwise a random name is used.
  • docker logs -f nginx-container - this will fetch the logs of a specified container. The -f will follow the logs and continue to stream new output from a container's STDOUT and STDERR
